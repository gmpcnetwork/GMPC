# GMPC

GMPC multi-chain structure distributed infrastructure

Website: https://www.gm-pc.com

## start up

1. install rust 

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

2. build project as release version

```
cargo build --release
```

3. run dev version


```
./target/release/gmpc
```
